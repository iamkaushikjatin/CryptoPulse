
   ```bash
   npm install
   ```

3. **Set Up Firebase**:
   - Configure Firebase in a `.env` file or `services/firebaseConfig.js` with your project credentials.
   - Also add the coinranking api in your `.env`.

4. **Run the Application**:
   ```bash
   npm run dev
   ```
